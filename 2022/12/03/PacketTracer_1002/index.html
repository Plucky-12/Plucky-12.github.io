动态路由OSPF配置——思科模拟器学习

Created2022-12-03|Updated2022-12-29
|Post View:

一、问题

假设校园网通过一台三层交换机连到校园网出口路由器上，路由器再和校园外的另一台路由器连接。现要做适当配置，实现校园网内部主机与校园网外部主机之间的相互通信。为了简化网管的管理维护工作，学校决定采用OSPF协议实现互通。

二、技术原理

OSPF开放式最短路径优先协议，是目前网路中应用最广泛的路由协议之一。属于内部网管路由协议，能够适应各种规模的网络环境，是典型的链路状态协议。OSPF路由协议通过向全网扩散本设备的链路状态信息，使网络中每台设备最终同步一个具有全网链路状态的数据库，然后路由器采用SPF算法，以自己为根，计算到达其他网络的最短路径，最终形成全网路由信息。

三、操作设备

PC 2台；Switch_3560 1台；Router-PT 2台；直连线；交叉线；DCE串口线

四、解决代码

如下图所示：

首先给每一台PC1和PC2配置IP和Submask
IP: 192.168.1.11
Submask: 255.255.255.0
Gateway: 192.168.1.1
注意:每台电脑都要配置（这里只配置了一台）！！！
其次配置三层交换机S3560
第1步：划分vlanin
Switch(config)#hostname xiaoyuan
xiaoyuan(config)#vlan 10
xiaoyuan(config-vlan)#exit
xiaoyuan(config)#vlan 20
xiaoyuan(config-vlan)#exit
第2步：将端口划分的相应的vlan中
xiaoyuan(config)#interface fa 0/10
xiaoyuan(config-if)#sw
xiaoyuan(config-if)#switchport acc
xiaoyuan(config-if)#switchport access vlan 10
xiaoyuan(config-if)#exit
xiaoyuan(config)#inter
xiaoyuan(config)#interface fa 0/20
xiaoyuan(config-if)#sw
xiaoyuan(config-if)#switchport acc
xiaoyuan(config-if)#switchport access vlan 20
xiaoyuan(config-if)#exit
第3步：配置三层交换机VLAN对应IP
xiaoyuan(config)#interface vlan 10
xiaoyuan(config-if)#ip address 192.168.1.1 255.255.255.0
xiaoyuan(config-if)#no shut
xiaoyuan(config-if)#exit
xiaoyuan(config)#inter
xiaoyuan(config)#interface vlan 20
xiaoyuan(config-if)#ip address 192.168.3.1 255.255.255.0
xiaoyuan(config-if)#no shut
xiaoyuan(config-if)#end
xiaoyuan#show vlan //查看一下配置成功没有

然后配置R1端口ip配置(配置其时钟频率64000)
xiaoyuan_r1(config)#interface fa 0/0
xiaoyuan_r1(config-if)#no shut
xiaoyuan_r1(config-if)#ip address 192.168.3.2 255.255.255.0
xiaoyuan_r1(config-if)#exit
xiaoyuan_r1(config)#inter
xiaoyuan_r1(config)#interface ser
xiaoyuan_r1(config)#interface serial 0/0/0
xiaoyuan_r1(config-if)#no shut
xiaoyuan_r1(config-if)#clock rat
xiaoyuan_r1(config-if)#clock rate 64000
xiaoyuan_r1(config-if)#ip add
xiaoyuan_r1(config-if)#ip address 192.168.4.1 255.255.255.0
xiaoyuan_r1(config-if)#end
xiaoyuan_r1#show ip rou//查看是否配置成功

R2端口ip配置：
xiaoyuan_r2(config)#interface fa 0/0
xiaoyuan_r2(config-if)#no shut
xiaoyuan_r2(config-if)#ip address 192.168.2.1 255.255.255.0
xiaoyuan_r2(config-if)#exit
xiaoyuan_r2(config)#inter
xiaoyuan_r2(config)#interface ser
xiaoyuan_r2(config)#interface serial 0/0/0
xiaoyuan_r2(config-if)#no shut
xiaoyuan_r2(config-if)#ip address 192.168.4.2 255.255.255.0
xiaoyuan_r2(config-if)#end
xiaoyuan_r2#show ip rou
show ip route //查看是否配置成功

动态RIP配置（三个路由设备都要进行配置）：
三层交换机动态RIP配置
三层交换机动态OSPF配置：
conf t
IP routing
router ospf 411 // 1-65535 Process ID（即100） 改成学号后三位
network 192.168.3.0 0.0.0.255 area 0
network 192.168.1.0 0.0.0.255 area 1
end

路由器R1动态RIP配置（路由器不用开启路由功能）：
xiaoyuan_r1(config)#router ospf 411
xiaoyuan_r1(config-router)#network 192.168.3.0 0.0.0.255 area 0
xiaoyuan_r1(config-router)#network 192.168.4.0 0.0.0.255 area 0
xiaoyuan_r1(config-router)#end
xiaoyuan_r1#show ip rou
路由器R2动态RIP配置：
xiaoyuan_r2(config)#router ospf 411
xiaoyuan_r2(config-router)#network 192.168.2.0 0.0.0.255 area 2
xiaoyuan_r2(config-router)#network 192.168.4.0 0.0.0.255 area 0
xiaoyuan_r2(config-router)#exit
xiaoyuan_r2(config)#exit
xiaoyuan_r2#show ip rou
