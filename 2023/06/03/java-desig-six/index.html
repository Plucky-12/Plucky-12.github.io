工厂方法模式✅

工厂方法是定义一个用于创建对象的接口,让子类决定实例化哪一个类。工厂方法将一个类的实例化延迟到其子类。

适配器模式（Adapter）包含以下主要角色：

抽象产品类：RAM 、CPU

具体产品类：PcRAM、MacRAM、 PcCPU、 MacCPU

抽象工厂类： ComputerAccessoriesFactory

具体工厂类：PcFactory、 MacFactory
